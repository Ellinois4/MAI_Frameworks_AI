# Лабораторные работы по курсу "Прикладные системы и фреймворки искусственного интеллекта"

## Выбор наборов данных и метрик качества

### Датасет для задачи классификации

Для задачи классификации был выбран датасет **[Realistic Loan Approval Dataset (US & Canada)](https://www.kaggle.com/datasets/parthpatel2130/realistic-loan-approval-dataset-us-and-canada/data)**

Данный набор данных моделирует процесс принятия решения о выдаче кредита в банковской сфере. Таргет — `loan_status`, принимающий бинарные значения (Approved / Rejected). Признаки включают демографические характеристики заёмщика, финансовые показатели, кредитную историю и параметры кредитного продукта.

Выбор данного датасета обоснован его практической значимостью: задача кредитного скоринга широко применяется в реальных финансовых организациях и является классическим примером бинарной классификации. 

### Датасет для задачи регрессии

Для задачи регрессии был выбран датасет **[Song Popularity Dataset](https://www.kaggle.com/datasets/yasserh/song-popularity-dataset/data)** 

Целевой переменной является числовой показатель `song_popularity`, отражающий уровень популярности музыкального трека. В качестве признаков используются различные аудиохарактеристики, например, энергичность, танцевальность, темп и акустичность.

Выбор данного датасета обусловлен его прикладной направленностью: прогнозирование популярности контента актуально для музыкальных стриминговых сервисов и рекомендательных систем. Задача регрессии в данном случае позволяет исследовать способность моделей аппроксимировать сложные нелинейные зависимости между характеристиками объекта и числовой целевой переменной.


### Выбор метрик качества и обоснование

Метрики для оценки качества моделей классификации:
- **Accuracy** — отражает долю верно классифицированных объектов;
- **Precision и Recall** — позволяют оценить качество классификации положительного класса;
- **F1-мера** — используется как компромисс между точностью и полнотой;
- **ROC-AUC** — характеризует способность модели различать классы независимо от выбранного порога.

Использование нескольких метрик позволяет получить более полное представление о качестве классификатора, особенно в условиях возможного дисбаланса классов.


Метрики для оценки качества моделей регрессии:
- **RMSE (Root Mean Squared Error)** — характеризует среднюю величину ошибки прогнозирования и чувствительна к большим отклонениям;
- **MAE (Mean Absolute Error)** — отражает среднюю абсолютную ошибку и является более устойчивой к выбросам;
- **R² (коэффициент детерминации)** — показывает долю объяснённой моделью дисперсии целевой переменной.

Выбранные метрики являются стандартными для задач регрессии и позволяют оценить как абсолютную точность прогнозов, так и общую объясняющую способность модели.
